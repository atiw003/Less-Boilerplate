Less Boilerplate

What's This?

I went through my recent design projects and pulled out the styles that I use all the time into an easy-to-import stylesheet. They help me get a new design started, and I hope they help you too!

It's written in Less, the dynamic stylesheet language.

What Does it Give Me?

  • CSS Reset
  • CSS3 Helpers (box shadow, gradients, box radius)
  • Centered column blocks
  • Horizontal centering for objects of variable width
  • A fancy call to action button
  • A few general styles

Documentation / Examples


CSS3 Helpers
Rounded Corners 
#element{.boxRadius(radius);}
.borderRadius(15px);

compiles to

border-radius: 15px;
-moz-border-radius: 15px;
-webkit-border-radius: 15px;
Gradients 
// Vertical
#element{.gradientV(startColor, endColor);}

// Horizontal
#element{.gradientH(startColor, endColor);}
.gradientV(@lavender, @red);

compiles to

background: #e3a6d8;
background: -moz-linear-gradient(top, #e3a6d8, #ff6464);
background: -webkit-gradient(linear, left top, left bottom, from(#e3a6d8), to(#ff6464));
filter: progid:DXImageTransform.Microsoft.gradient(startColorstr=#e3a6d8, endColorstr=#ff6464);
-ms-filter: progid:DXImageTransform.Microsoft.gradient(startColorstr=#e3a6d8, endColorstr=#ff6464);
.gradientH(@lavender, @red);

compiles to

background: #e3a6d8;
background: -moz-linear-gradient(left, #e3a6d8, #ff6464);
background: -webkit-gradient(linear, left center, right center, from(#e3a6d8), to(#ff6464));
filter: progid:DXImageTransform.Microsoft.gradient(startColorstr=#e3a6d8, endColorstr=#ff6464, GradientType=1);
-ms-filter: progid:DXImageTransform.Microsoft.gradient(startColorstr=#e3a6d8, endColorstr=#ff6464, GradientType=1);
Drop Shadow 
#element{.boxShadow(xOffset, yOffset, blur, color);}
.boxShadow(3px, 3px, 6px, #000);

compiles to

-moz-box-shadow: 3px 3px 6px #000000;
-webkit-box-shadow: 3px 3px 6px #000000;
box-shadow: 3px 3px 6px #000000;
Stacking & Centering

For designs that involve a centered column of fixed with, I've found that for that for great browser compatability, it's useful to separate page elements into discreet horizontal chunks when possible. The classes wrapOut and wrapIn help handle this.

Each section on this page uses these two styles. 
<div class="wrapOut">
  <div class="wrapIn">
    Content goes here, is centered, and stacks nicely.
  </div>
</div>
<div class="wrapOut">
  <div class="wrapIn">
    Same here.
  </div>
</div>

Usage

Import the Less Boilerplate stylesheet at the top of your main stylesheet: 

@import 'boilerplate';
